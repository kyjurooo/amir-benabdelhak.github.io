## 🚀 Mise en ligne sur GitHub Pages (gratuit)

### Étape 1 — Créer un compte GitHub
Va sur https://github.com et crée un compte (si pas déjà fait).

### Étape 2 — Créer un repository
- Clique sur "New repository"
- Nom : `amir-benabdelhak.github.io` (exactement ce format)
- Coche "Public"
- Clique "Create repository"

### Étape 3 — Uploader les fichiers
- Clique "uploading an existing file" sur la page du repo
- Glisse-dépose TOUS les fichiers du dossier portfolio (+ le dossier images/)
- Clique "Commit changes"

### Étape 4 — Activer GitHub Pages
- Va dans Settings → Pages
- Source : "Deploy from a branch" → main → / (root)
- Sauvegarde

### Résultat
Ton site sera accessible sur : **https://amir-benabdelhak.github.io**

(Peut prendre 2-3 minutes à la première mise en ligne)
